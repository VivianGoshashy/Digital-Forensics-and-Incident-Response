# Digital-Forensics-and-Incident-Response
Repository: Digital Forensics Labs & Projects
Course: SEC 350HB – Digital Forensics (Winter 2026)
Instructor: Evan Drake
Institution: Bellevue College

This repository contains labs, projects, and forensic analysis exercises completed as part of the SEC 350HB Digital Forensics course. The coursework focuses on the fundamentals of computer forensics, legal considerations, incident response, and hands-on forensic investigation across multiple platforms.

Topics and Tools Covered:

Forensic imaging and analysis using industry-standard tools
Windows and Linux forensic investigation techniques
Network intrusion detection and analysis
Legal and ethical considerations in digital forensics
Report writing based on forensic evidence
Use of VirtualBox, NDG Forensics v2, and WWU Cyber Range environments
Exploration of anti-forensics and digital privacy concepts
Key Skills Developed:

Conducting full computer forensic examinations
Applying incident response best practices
Analyzing forensic data on Windows and Linux systems
Evaluating recovery tools and network attack software
Writing clear, accurate forensic reports
Course Text:
Practical Linux Forensics: A Guide for Digital Investigators by Bruce Nikkel

Note:
All work in this repository is for educational purposes and reflects the assignments and learning outcomes of the course. Proper credit is given to sources and tools used, in alignment with the course’s AI and academic integrity policies.
